## 🤝 Katkı Sağlama
1. Yeni bir çalışma için ana dizinde klasör açın.
2. Değişikliklerinizi kendi branch'inizde yapın.
3. `main` branch'ine **Pull Request** açarak ekip liderinin onayını bekleyin.
